Instalar en cPanel

1.- Subir el sistema en archivo ZIP en la ruta deseada, puede ser una carpeta o subdominio.

2.- Descomprimir el ZIP con las opciones del cPanel

3.- Crear usuario y base de datos en MySQL y luego vincular el usuario a la base de datos.

4.- Importar el fichero "Gesnet.sql" en la base de datos que se desea trabajar. Sin embargo, en archivo se encuentra en la carpeta "bd/Gesnet.sql"

5.- Editar el Fichero "model/Conexion.php", en este archivo se tiene que colocar las credenciales de MySQL creadas en el punto 3.

6.- Acceder al sistema via URL

7.- Usuario: admin Contraseña: 12345
